About Us

BlitzBand is a demo brand that was created by Ottemo for Developers to provide an example of an eCommerce store running Ottemo in production. BlitzBand is not a real brand, but included is a demo catalog of wearable health-tracking monitor bands from Nike, FitBit, and Garmin. The Ottemo offices are located directly across the CenturyLink Field, the home of the Seattle Seahawks, whose mascot is called Blitz, hence the name.

The entire store, from the Store and Dash, to the backend database management and payment processing, are all written in-house at Ottemo. The store homepage, catalog, product page, are all run using the Ottemo Store-ng. The Dash administrator panel is run using the Ottemo Dash. If you would like to customize or contribute to either projects, please visit us on GitHub.

If you would like to learn more about Ottemo, please visit us at Ottemo.io.
